# jdk镜像站

请请求`index.json`谢谢

返回结果为：
```json
{
    "mirrors":{
        "<操作系统名称>":[
            {
                "name":"<jdk名称>",
                "type":"<jdk类型（arm64、x64）>",
                "url":"<镜像地址>"
            }
        ]
    }
}
```
因为GitHub太慢（只是对于无VPN的用户来说），使用`URL`为**华为镜像站**的URL<br>
